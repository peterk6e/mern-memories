# MERN MEMORIES

Web app to share your memories with your friends.

Project using: 
- MongoDB
- Express
- React
- Node
- Redux
- JWT
- Google API
- Material-UI
- React router

After registration, the user can create edit posts, he can like, comment, search by title or by tags
